# Ford GoBike Data Visualization

Exploratory and explanatory data visualization project using Ford GoBike trip data from February 2019.

## Project Overview

This project is divided into two parts:

- Part I: Exploratory data analysis
- Part II: Explanatory data visualization presentation

## Key Findings

- Subscribers represent the majority of rides.
- Bike usage peaks during weekday commuting hours.
- Customers tend to take longer rides than Subscribers.

## Files

- `Part_I_notebook.ipynb` - exploratory analysis notebook
- `Part_I_notebook.html` - exported exploratory analysis
- `Part_II_notebook.ipynb` - explanatory presentation notebook
- `Part_II_notebook.html` - exported explanatory presentation
- `201902-fordgobike-tripdata (1).csv` - dataset
